Read Me

Install Python Version 3.13 -  for more information please refer to this website : https://www.python.org/downloads/release/python-3130/

Pull our repository.

Open the file in Visual Code.

Open a terminal and install PySide and My-sql-connector by entering the following:

pip install PySide6
pip install mysql-connector-python==8.0.20

Then run main.py by using the following command while inside the PAMS directory: 

python main.py
